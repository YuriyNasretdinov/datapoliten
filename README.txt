Описание структуры файлов и папок:

README.txt -- описание структуры файлов и папок (вы сейчас это читаете)

pub/       -- для любых файлов, доступных напрямую из web, как http://.../pub/...
pub/i/     -- сюда класть только изображения!
pub/css/   -- для css-файлов
pub/js/    -- для js-скриптов и библиотек 
(если библиотеки идут со своими картинками и стилями, это не запрещается)
pub/mix/   -- всё, что не попадает под категории выше

core/      -- ядро системы
core/lib/  -- библиотеки, предназначенные для сервера

templates/ -- шаблоны страниц и отдельных частей страниц

dolphin/ -- файловый менеджер "дельфин" (удобно, надежно, быстро :))
pma/     -- PhpMyAdmin, средство для управления базами данных MySQL

files*/  -- папки пользователей с файлами на разных винчестерах

converted/ -- папка с символическими ссылками на папки .convert пользователей (чтобы можно было легально просматривать .html-файлы)
