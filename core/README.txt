Описание структуры core/:

config.php -- основной файл конфигурации системы

lib/       -- библиотеки, все файлы на старте включаются
modules/   -- модули, которые необязательно будут все подключены на старте

Общие рекомендации по стилю оформления кода:

названия классов: английские, начинаются с Большой Буквы
названия методов и функций: английские, маленькими буквами, с_подчеркиванием

слова email, password, и другие должны писаться _целиком_ везде, без сокращений, вроде "mail", "pass" и др;
