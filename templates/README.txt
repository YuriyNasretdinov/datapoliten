описание файлов в templates/:

index.html    -- главная страница / список файлов (после входа)
login.html    -- вход в систему
profile.html  -- профиль пользователя
register.html -- регистрация нового пользователя
upload.html   -- загрузить файлы

indirect/     -- шаблоны, не вызывающиеся напрямую набором http://host/some.html, и данные берутся с предварительной обработкой на PHP
service/      -- для системных файлов, которые используются для шаблонов, которые включаются с помощью include()
mail/         -- для шаблонов писем
